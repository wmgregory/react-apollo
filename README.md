## Getting Started

### Install dependencies

In terms of installing the webapp, this is all you need to do:

```
$ cd client/ && yarn
$ cd server/ && yarn
```

### serve

To run the server (through AWS)

```
$ yarn serve
```

## Running Tests

Coverage is improving gradually!

### Run unit tests

Uses Jest + Enzyme

```
$ yarn test          # local
```

### Run E2E tests

Uses Cypress

```
$ yarn e2e           # local GUI
```
